# Log Achievements

When a significant professional milestone completes during a conversation — a PR is created or merged (`gh pr create`, `gh pr merge`), a design doc is finalized, or an important ad-hoc task wraps up — proactively offer to log it in one short line:

> "Want me to log this as an achievement? `/log-achievement`"

Do this once per milestone, non-intrusively. Don't ask for routine, small, or repetitive tasks.
